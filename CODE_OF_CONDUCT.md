# Code of Conduct

Fundcloud adopts the [Contributor Covenant, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). In short: be kind, assume good faith, and focus on the technical merits of contributions. Harassment, personal attacks, and discrimination are not tolerated.

Report violations by [opening a GitHub issue](https://github.com/cyberapper/fundcloud/issues/new?labels=conduct&title=Code+of+Conduct+concern). Reports are handled confidentially.
