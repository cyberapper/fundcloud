# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x | ✅ |
| < 0.1 | ❌ |

## Reporting a vulnerability

Please **do not** open a public GitHub issue.

Use GitHub's [private vulnerability reporting](https://github.com/cyberapper/fundcloud/security/advisories/new) — no email needed. Please include:
- A description of the issue and its impact.
- Reproduction steps or a proof-of-concept, if possible.
- Any mitigation you've applied locally.

We aim to acknowledge receipt within 3 business days and to ship a fix within 30 days for critical issues.
